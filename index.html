🔧 Cloudflare Worker Setup
One-time setup to enable cloud sync for the whole team.
1
Create a Cloudflare account
cloudflare.com → Workers & Pages → KV
2
Create a KV Namespace
Name it TA_RECORDS. Note the Namespace ID.
3
Deploy the Worker
Workers & Pages → Create Worker → paste code below → bind KV as TA_RECORDS → Deploy.
4
Set environment variable
Add SENDER_EMAIL = your verified sender (e.g. noreply@yourdomain.com) in Worker Settings → Variables.
